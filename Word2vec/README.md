# Word2vec
Implementation of models to produce word embeddings

## Architecture

1. Skip-gram

## Libraries used

1. NumPy (1.11.2)
2. Matplotlib (1.5.1)
3. Tensorflow (0.11)
4. Scikit-learn (0.18)

## How to run

1. Place **text8.zip** dataset into dataset folder.
2. Run **word2vec.py**.
